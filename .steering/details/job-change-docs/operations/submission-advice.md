# 提出・応募アドバイスルール

## 目的

作成した転職書類を効果的に活用するための応募戦略・提出方法・
面接対策について推奨事項を提供する。

---

## 提出先・応募チャネルの判断

| チャネル | 向いているケース |
|---------|----------------|
| **転職エージェント** | 年収交渉・非公開求人・書類添削サポートを求める場合 |
| **直接応募（企業HP）** | ターゲット企業が明確で強い志望度がある場合 |
| **転職媒体（LinkedIn等）** | 複数企業への効率的なアプローチ・スカウト活用の場合 |
| **リファラル（知人紹介）** | 社内文化・実態の事前把握・選考通過率向上が目的の場合 |

シニアエンジニア・テックリード相当のポジションは、エージェント経由の非公開求人または
ダイレクトリクルーティングが有効なケースが多い。

---

## 提出前の最終確認

提出前に以下をチェックする：

- [ ] 事実と異なる記述・誇張がないか
- [ ] 社外秘・機密情報（顧客名・売上数値等）を無断で記載していないか
- [ ] 日付・在籍期間の整合性が取れているか
- [ ] 応募先企業名で検索したとき他社名が混在していないか（企業別カスタマイズ版の場合）
- [ ] ファイル名・フォーマットが応募先の指定に合っているか

---

## 面接対策への接続

書類作成で整理した情報を面接準備に活用する：

| 書類の要素 | 面接での活用 |
|-----------|------------|
| STAR形式エピソード | 行動面接（Behavioral Interview）への回答原稿として活用 |
| 訴求メッセージ | 「自己紹介」「志望動機」の骨格として活用 |
| 差分・ギャップの補完表現 | 「弱みをどう補うか」の回答として活用 |
| ターゲット分析の適合ポイント | 「なぜこの会社か」の回答根拠として活用 |

---

## フォローアップ

| タイミング | アクション |
|-----------|-----------|
| 応募後1週間以内 | エージェント経由の場合は進捗確認を依頼する |
| 書類通過後 | 面接で深堀りされそうな実績エピソードをSTAR形式で再整理する |
| 不合格の場合 | フィードバックを収集し、次回応募時の書類改善に反映する |
| 内定後 | 年収・入社条件の交渉に書類の訴求ポイントを活用する |

---

## 出力

アドバイスを以下の形式で提示する：

```
【応募戦略のご提案】

1. おすすめのチャネル：[チャネル名] — [理由]
2. 提出前の確認ポイント：[重要チェック項目]
3. 面接対策への活用：[書類との接続ポイント]

ご質問や特定企業への応募準備についても対応可能です。
```
